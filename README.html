GRASS GIS binaries for MS Windows

URL: grass.osgeo.org

Supported platforms:

  • x86_64 - 64bit binaries
  • x86 - 32bit binaries

Content:

  • osgeo4w - OSGeo4W package
  • addons - GRASS Addons
  • logs - Build logs
  • WinGRASS*.exe - Standalone installer

Binaries are generated daily.
